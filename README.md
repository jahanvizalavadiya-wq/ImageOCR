This project is an Image OCR (Optical Character Recognition) application that allows users to upload images and extract text from them. It is built using React.js for the frontend and ASP.NET Core Web API for the backend. The application demonstrates how to process images, extract readable text, and display it to users efficiently.

Features:

Upload images (PNG, JPG, JPEG)
Extract text from uploaded images
Display extracted text in a user-friendly format

Frontend: React.js, HTML, CSS, JavaScript
Backend: ASP.NET Core Web API (C#)
OCR Library: Tesseract OCR or any chosen OCR engine
Others: Axios (for API requests), FormData (for file uploads)
